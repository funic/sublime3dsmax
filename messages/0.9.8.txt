Use Sublime for MAXScript Development with 3ds Max.


Website:

    https://github.com/cb109/sublime3dsmax


Changelog:

    0.9.8
        - bug  : Fix autocompletion from .sublime-package

    0.9.7
        - feat : Add command 'select_max_instance' to choose a 3ds Max
                 to talk to when multiple are available
        - feat : Change selection behaviour, single line selections are
                 now send to 3ds Max exactly as they are without expansion
        - feat : Expose all commands via command palette
        - bug  : Fix Python scripts failing to execute
        - drop : Remove Py3dsMax compatibility

    0.9.2
        - feat : Add command to browse the online maxscript help for the
                 currently selected maxscript code.

    0.9.0
        - feat : Add autocompletion for file content and maxscript API.
        - feat : Add snippets for common code blocks like if-do, if-then-else,
                 function, rollout, struct etc.

    0.8.4
        - feat : Add compatibility for 3ds Max 9.
        - feat : Add compatibility for Py3dsMax.

    0.8.2
        - feat : Add tags for automatic update propagation.
        - feat : Add install and update messages.
        - feat : Read block comment support and add spaces in between
                 the block start and end tokens.
        - bug  : Fix missing color coding for > and < operators.

    0.8.1
        - feat : Make return value of filein appear on new line.
